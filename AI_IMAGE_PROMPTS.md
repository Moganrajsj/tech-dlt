## Tips for Best Results

### For Midjourney:
- Add `--v 6` for latest version
- Use `--style raw` for more photorealistic results
- Add `--ar 1:1` for square format
- Try `--q 2` for higher quality

### For Leonardo.ai:
- Use "PhotoReal" or "3D Render" model
- Enable "Alchemy" for better quality
- Set dimensions to 1024x1024
- Adjust "Prompt Magic" strength to 0.5-0.7

### For Bing Image Creator:
- Use the full detailed prompt
- Request "photorealistic 3D render"
- Specify "dark background"
- Download the highest resolution available

### For Stable Diffusion:
- Use models like: Realistic Vision, DreamShaper, or SDXL
- Negative prompt: "cartoon, anime, illustration, painting, drawing, low quality, blurry"
- CFG Scale: 7-9
- Steps: 30-50
- Sampler: DPM++ 2M Karras
